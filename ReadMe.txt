You can run the plugin by loading the LSystemd.mll from the project folder then go to one of the menus.
1.When you go to LSystemCmd menu you will get a window using which the grammar, iterations stepsize and angle can be changed. 
  Before clicking on create make sure you have a nurbs circle created.
2.When you go to LSystemNode then just change the attribute values in the attribute editor. 
  For grammar just pass the grammar string from text files directly.
